# 🌐 BlogSphere – Modern Blogging Platform

A clean and responsive blogging platform where users can create, read, update, and delete stories.
Built with a focus on simplicity, readability, and a polished user interface.

---

## 🚀 Live Features

* ✍️ Create, edit, and delete blog posts
* 🔐 User authentication (Login / Register)
* 🧾 Category-based filtering (Technology, Travel, Lifestyle, etc.)
* ❤️ Like feature on blogs
* 📖 Read time estimation
* 🎯 Featured blog section
* 📱 Fully responsive design
* 🎨 Clean UI with modern typography and image-based cards

---

## 🛠️ Tech Stack

### Frontend

* HTML
* CSS (Custom styling, no frameworks)
* JavaScript (Vanilla JS)

### Backend

* Node.js
* Express.js

### Database

* MongoDB

### Authentication

* JWT (JSON Web Tokens)

## ⚙️ Setup Instructions

### 1. Clone the repository

```
git clone https://github.com/your-username/BlogSphere.git
cd BlogSphere
```

---

### 2. Setup Backend

```
cd backend
npm install
```

Create a `.env` file:

```
PORT=5000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
```

Run server:

```
npm run dev
```

---

### 3. Run Frontend

Just open in browser:

```
http://localhost:5000
```
